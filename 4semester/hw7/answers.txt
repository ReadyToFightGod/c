1) Каким образом происходит передача информации между устройствами в сети Интернет?
У каждого устройства есть свой уникальный адрес (ip). Устройство через кабель или модем посылает запрос на сервер, этот запрос 
перемещается между роутерами (как посылка с шейна по отделениям почты россии) до тех пор, пока не достигнет сервера (если правильно помню, то
таких перемещений может быть максимум где-то около двадцати, иначе запрос удаляется // выкидывается из сети и надо отправлять по новой). Сервер получает запрос, 
обрабатывает и отправляет ответ на нужный ip адрес (то есть на посылке пишется нужный ip и ее опять кидают в сеть с роутерами, которые
должны направить ее)

2) Почему передача больших объёмов данных (например, архива с датасетом для машинного обучения) по Интернету не задерживает 
обмен данными между другими устройствами в сети?
Потому что передача маленькими порциями, а не сразу. Поэтому в промежутке между маленькими порциями одного файла могут проскочить
данные для других устроств.
